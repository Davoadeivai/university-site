# 🚂 Railway Deployment Guide - README

## 🎯 Quick Start

این پروژه برای **Railway** آماده شده است.

### ✅ تمام فایل‌های مورد نیاز:
```
✓ config/settings.py          - Production settings
✓ requirements.txt             - Dependencies
✓ Procfile                     - Railway config
✓ build.sh                     - Build script
✓ runtime.txt                  - Python 3.11.9
✓ .env.example                 - Environment template
✓ .gitignore                   - Git rules
```

### 🚀 سه مرحله:

1. **Railway Account** → https://railway.app
2. **GitHub Connect** → Select `university-site`
3. **Deploy** → تنظیم Environment Variables

### 📖 راهنمای کامل:
```
RAILWAY_SETUP_STEPS.md
```
